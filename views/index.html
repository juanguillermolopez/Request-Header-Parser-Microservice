Request Header Parser Microservice

How to Use

This API returns information about your request headers, including your IP address, language, and software.

Example Usage:

GET /api/whoami
{ "ipaddress": "159.20.14.100", "language": "en-US,en;q=0.5", "software": "Mozilla/5.0 (X11; Ubuntu; Linux x86_64; rv:50.0) Gecko/20100101 Firefox/50.0" }

API Endpoints

GET /api/whoami

Returns information about your request:

  • ipaddress: Your IP address
  • language: Your preferred language(s)
  • software: Your browser/operating system

GET /api/requests

Returns the last 50 requests (for testing)

GET /api/stats

Returns statistics about requests
